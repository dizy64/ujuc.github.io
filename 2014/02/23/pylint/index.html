    • pylint 홈페이지

    설치

    pip install pylint

    사용

    pylint filename.py
pylint-gui
    • GUI를 사용하게 되면 쉽게 확인이 가능하다.
    • 콘솔에서 표기되는 내용들을 탭별로 보여준다.

    좋은점

    • IDE를 사용해서 코드를 작성하면 Python PEP8에 작성된 내용과 비슷하게는 되는데 못잡아주는 것도 있어서 이걸 사용하면된다.
    • 모듈 이름도 가이드해주더라.
    • 함수내에 뭔짓을 이렇게 많이 하고 있느냐도 물어봐주고
    • 코드를 작게 줄일 수 있도로 도와주는 역활도 함.
